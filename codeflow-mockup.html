왜 CodeFlow인가요?

복잡한 디버거 없이, 코드가 어떻게 동작하는지 직관적으로 이해합니다.

🧠

라인별 실행 시각화

코드가 한 줄씩 실행될 때 무슨 일이 일어나는지 단계별로 추적합니다.

📦

Stack / Heap 시각화

변수와 객체가 메모리에서 어떻게 저장되고 참조되는지 보여줍니다.

🔁

재귀 흐름 시각화

Call Stack 구조를 시각적으로 표현해 재귀 함수의 동작을 이해합니다.

🎯

포인터 / 배열 시각화

투포인터와 배열 인덱스 이동을 화살표로 직관적으로 표현합니다.

▶

Step 실행 모드

한 줄씩 수동으로 실행하며 각 단계에서의 상태를 직접 확인합니다.

⚠️

에러 위치 강조

오류가 발생한 정확한 라인과 원인을 시각적으로 즉시 파악합니다.
